Welcome to Truecaller Clone API

I have implemented both global search by phone and name !!!

INSTRUCTIONS to run

  • download all requirements listed in requirements.txt

  • install postgres database if not installed

  • create a database and edit the 'DATABASE' property in 'settings.base.py'

  • python manage.py makemigrations

  • python manage.py migrate

  • python manage.py runserver

  • Create dummy users and multiple contacts as mentioned below

  • Go to 'api_usage' folder and follow instructions to use API

Custom Commands to add dummy_data

  • Add dummy user
    python manage.py dummy_user 'username' 'name' 'password' 'email'
    ex:python manage.py dummy_user "9995477921" "sunil23" "sunil1234" "sunilpie@sunil.com"

  • Add multiple dummy phone_numbers to phone directory at once
    python manage.py dummy_contacts 'contacts'
    ex:python manage.py dummy_contacts "2222222222" "3333333333" "4444444444" "5555555555"

  • Adds multiple phone no's to spam.If phone no exists,increase spam score
    python manage.py add_to_spam 'contacts'
    python manage.py add_to_spam "4444444444" "5555555555"
